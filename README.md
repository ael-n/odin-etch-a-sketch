# odin-etch-a-sketch

A dynamic grid drawing tool where you can create colorful patterns by hovering over cells. Each cell starts with a random color and gradually becomes more visible as you hover over it.

## Features

- **Dynamic Grid Size**: Create grids from 1x1 up to 100x100 cells
- **Colorful Experience**: Each cell has a random background color
- **Progressive Reveal**: Cells become more opaque with each hover (10% increments)
- **Responsive Design**: Fixed 600x600px container that adapts to any grid size

## How to Use

1. Click the button to create a new grid
2. Enter a number between 1 and 100 for grid size
3. Hover over any cell to reveal its color
4. The more you hover, the more visible the cell becomes

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
